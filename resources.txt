The charts in dataset DS-Tbl has been created using the matplotlib library for the csv files taken from the follwing resources:

1. For histograms and scatterplots:
https://vincentarelbundock.github.io/Rdatasets/datasets.html

2. For bar chart:
https://people.sc.fsu.edu/~jburkardt/data/csv/csv.html

DS-Img folder contains chart images that has been downloaded from web or scanned from documents. The web link for the images are as following:
DS-Img-sp-1: https://www.siyavula.com/read/maths/grade-11/statistics/11-statistics-06
DS-Img-sp-2: https://www.ncl.ucar.edu/Applications/Images/scatter_2_1_lg.png
DS2-hg-4: http://bpg.oicr.on.ca/API/BoutrosLab.plotting.general/5.3.4/create.histogram.html
DS-Img-gbc-1: https://drive.google.com/file/d/1VW5VN514lVRvGweNe7VLKf706sZLHL2e/view
DS-Img-bc-2: http://www.datasciencemadesimple.com/r-bar-chart/
DS-Img-bc-1: https://www.listendata.com/2019/06/matplotlib-tutorial-learn-plot-python.html


DS-Img-bc2 bar chart is scanned image of the printout of the given web page.
